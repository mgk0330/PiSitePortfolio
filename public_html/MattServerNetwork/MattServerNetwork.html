Networking

Currently, my home network consists of a modem, and 2 routers servers a switch nodes.


Modem:

2 currently used and active ports. Both feeding to the router/switches for additional functionality.


Router 1:

Serves as the main wireless hub and primary LAN connection point for the network. This works to reduce lag time between internetwork devices. The media server, my pc, my wife's pc, and our 8 Wi-Fi devices all run through this node.


Router 2:

Due to not having gigbit LAN support, this router currently only plays host to the webserver, however it has Wi-Fi capabilities and additional ports so expansion is possible if needed.
